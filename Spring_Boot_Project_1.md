# Spring Boot Project 1 - Student Management System

## Project Repository
[Student Management Spring Boot Project](https://github.com/AllenLeiZhao/spring-boot-project-hw)

## Project Overview
A RESTful API built with Spring Boot for managing students.

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok
- Maven

## API Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| POST | /api/students | Create student |
| GET | /api/students | Get all students |
| GET | /api/students/{id} | Get student by id |
| PUT | /api/students/{id} | Update student |
| DELETE | /api/students/{id} | Delete student |
